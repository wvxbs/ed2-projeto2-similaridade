Estruturas de dados são fundamentais para a ciência da computação. Uma árvore AVL é uma árvore binária de busca balanceada onde a diferença de altura entre as subárvores esquerda e direita de qualquer nó é no máximo um. O balanceamento é mantido através de rotações simples e duplas realizadas após inserções e remoções. A tabela hash é uma estrutura que mapeia chaves para valores utilizando uma função de dispersão. A resolução de colisões pode ser feita por sondagem linear ou encadeamento. Estruturas de dados eficientes são essenciais para algoritmos de alto desempenho.
